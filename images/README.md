# Images Directory

Place your flight training photos in this directory.

## Required Images:

1. **flight1.jpg** - Flight training session photo
2. **flight2.jpg** - Cockpit view or training aircraft
3. **flight3.jpg** - Scenic flight or aerial view
4. **flight4.jpg** - Student success or aviation-related photo

## Image Specifications:

- **Format**: JPG, PNG, or WebP
- **Recommended size**: 1200x800 pixels (or similar 3:2 aspect ratio)
- **File size**: Keep under 500KB for optimal loading speed
- **Orientation**: Landscape orientation works best

## Note:

The website currently uses fallback images from Unsplash. Once you add your own photos with the names above, they will automatically replace the placeholder images.
